# FDB-Proyecto
Sistema de Gestión de Arrendamientos
Este proyecto consiste en el diseño e implementación de una base de datos relacional para
optimizar la administración de rentas de departamentos. El sistema permite gestionar de
manera centralizada la información de los inquilinos, los detalles técnicos y disponibilidad de
los inmuebles, la formalización de contratos y el historial detallado de pagos mensuales.

El objetivo es sustituir procesos manuales por una estructura digital que garantice la integridad
de los datos y facilite la generación de reportes financieros.
